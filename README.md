# My-Spotify

My-Spotify mimics the Spotify interface, providing a visually appealing music player experience. It features a sidebar for navigation, a main content area for displaying music albums, and a fixed bottom music player for seamless playback control.

## Features

- **User-Friendly Navigation:** Intuitive sidebar for easy access to different sections.
- **Music Player Controls:** Includes play, pause, and volume adjustments.
- **Card-Based Layout:** Attractive display for music albums and playlists.
- **Modern Design:** A clean, dark-themed aesthetic inspired by popular music platforms.

## Technologies Used

- HTML
- CSS

## Getting Started

To run the project locally, clone the repository and open the `index.html` file in your preferred web browser.
